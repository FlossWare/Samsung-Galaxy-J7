# Where to Update GitHub Repository Description

## Exact Location on GitHub

1. **Go to your repository:**
   https://github.com/FlossWare/Samsung-Galaxy-J7

2. **Look at the RIGHT SIDE of the page** (or top on mobile)
   - You'll see a section called **"About"**
   - Currently shows: "Setup for my old Galaxy J7, second phone"

3. **Find the gear icon (⚙️)**
   - It's to the RIGHT of the "About" heading
   - Small gray gear icon
   - May say "Edit repository details" on hover

4. **Click the gear icon (⚙️)**
   - A modal/popup window will appear
   - Title: "Edit repository details"

5. **In the popup:**
   - **Description** field (first field) - This is what you want to update
   - Website field (optional)
   - Topics field (add tags here)
   - Options checkboxes (Releases, Packages, Deployments, Environments)

6. **Update the Description field:**
   - Delete: "Setup for my old Galaxy J7, second phone"
   - Paste: One of the suggested descriptions (see below)

7. **Click "Save changes"** (green button at bottom)

## Can't Find It?

If you're on **mobile**, the About section might be collapsed:
- Tap the repository name at the top
- Scroll down to find "About"
- Tap the three dots (•••) to edit

If you're on **desktop** and don't see it:
- The About section is on the right sidebar
- Make sure you're on the main repository page (Code tab)
- Not in Settings, Issues, or Pull Requests

## Alternative: Update via Repository Settings

If you still can't find it, you can also update from Settings:

1. Go to repository **Settings** tab (top menu)
2. Scroll down to the **"Options"** section
3. Under **"Repository name"**, you'll see **"Description"**
4. Update the description there
5. Scroll down and click **"Update settings"** or it auto-saves

---

**Note:** You must be logged into GitHub and have write access to the repository.
